About:

This project aims to model sleep disorders based on various lifestyle factors such as age, gender and BMI

This projects uses a Supervised Machine Learning algorithm (Random Forest) to predict and identify sleep disorders such as sleep apnea and insomnia.

It uses existing data and data analysis techniques to categorize sleep-related metrics.

KEY FEATURES:

Automated prediction of sleep disorders based on lifestyle factors

Uses comprehensive sleep data, lifestyle factors and cardiovascular health indicators

Enhances and validates existing sleep disorder prediction measures through AI measures
